PRD-Reader 简介

✿ 交互说明

这是一个高度定制化的阅读器，本次原型重点展示分享功能：

  • 鼠标悬停在左侧 docs 或 readme.md 上，点击出现的 图标。
  • 体验圆润风格的 Popover 气泡菜单。
  • 点击菜单项，体验弹性的 Toast 反馈。
  • 点击右上角的按钮，体验与系统高度融合的 404 缺省状态页。

仓库无权限或未配置

您访问的分享链接指向了一个未被系统授权的仓库。请联系管理员添加白名单。

文件不存在或已被删除

您访问的文档路径在此仓库中无法找到。它可能已被重命名、移动或删除。
