# 功能清单 (Feature List)

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| **交互式 PRD 模板** | **原型独立查看入口** | 在右侧 iframe 原型展示容器的右上角，提供一个常驻半透明的“新标签页打开”悬浮图标。 | 高 |
| **交互式 PRD 模板** | **悬浮交互反馈** | 当鼠标悬停在“新标签页打开”图标上时，图标变为完全不透明，并弹出 Tooltip 提示“在新标签页中打开原型”。 | 中 |
| **交互式 PRD 模板** | **URL 同步解析** | 点击图标后，不读取不可靠的 iframe `src`，而是解析当前外层 URL Hash 维护的激活状态，拼接为对应的原型文件相对路径。 | 高 |
| **交互式 PRD 模板** | **新标签页打开** | 调用 `window.open`，在浏览器的新标签页中完整加载该 HTML 文件，不携带左侧的文档和 TOC。 | 高 |
