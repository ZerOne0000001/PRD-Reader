# 功能清单梳理 (Feature_List) - F2_仓库切换

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| **全局路由层** | URL 参数拦截与导流 | 根据 URL 是否包含 `?repo=` 参数，决定是渲染“独立仓库列表页”还是“Reader 页面”。处理旧链接（无 repo 有 file）的重定向降级逻辑。 | P0 |
| **独立列表页** | 仓库数据读取 | 读取系统配置的白名单仓库列表作为页面数据源。 | P0 |
| **独立列表页** | 无数据空状态 | 当白名单为空时，展示友好的空状态占位图及提示信息。 | P1 |
| **独立列表页** | 仓库卡片网格渲染 | 以响应式网格的形式渲染所有可用的白名单仓库卡片。 | P0 |
| **独立列表页** | 仓库卡片跳转交互 | 点击仓库卡片，跳转至阅读器，URL 附带对应的 `?repo=xxx` 参数，并清除旧的 file 参数。 | P0 |
| **阅读器 (Reader)** | 单一仓库视图限制 | 调整 Reader 的文件树加载逻辑，强制仅拉取并展示当前 URL 中 `?repo=` 参数指定的单一仓库的文件树。 | P0 |
| **阅读器 (Reader)** | 头部仓库名称展示 | 将左上角原本的全局标题替换为当前正在阅读的仓库名称，并增加表示可下拉的视觉提示（如下拉箭头）。 | P1 |
| **阅读器 (Reader)** | 仓库切换下拉菜单 | 点击左上角仓库名称，弹出一个轻量级的下拉菜单（Dropdown），列出所有可用的白名单仓库。当前选中的仓库需要有高亮/Check 标识。 | P0 |
| **阅读器 (Reader)** | 下拉列表项：切换仓库 | 在下拉菜单中点击其他仓库，更新 URL 的 `?repo=` 参数，清空 `?file=`，触发状态机重置并重新拉取新仓库的文件树，展示 Loading 状态。 | P0 |
| **阅读器 (Reader)** | 下拉列表项：返回列表页 | 在下拉菜单的底部提供一个明显的“返回所有项目”或“返回选仓库页面”的按钮/链接，点击后清除所有参数跳转回首页。 | P0 |
