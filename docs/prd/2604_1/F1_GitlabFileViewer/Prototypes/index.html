系统架构设计

欢迎阅读核心文档！本文档描述了 PRD Reader 的技术架构。它被设计为一个纯前端代理平台，旨在提供极速的跨项目文档阅读体验。

核心流程

前端向代理服务发起请求。代理服务使用配置的 全局 Token 安全地请求 GitLab API，获取文件流并返回给前端进行优美的渲染。

✨

数据流向

用户 代理服务 GitLab

技术栈

使用以下优秀的工具构建：

React 18 + Vite
Tailwind CSS
marked.js
mermaid.js
