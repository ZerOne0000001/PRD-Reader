# 功能清单 (Feature List)

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| **后台配置** | 可见仓库白名单 | 管理员可配置允许前端访问的 GitLab 仓库列表（支持通过项目 ID 或路径配置）。 | P0 (核心依赖) |
| **后台配置** | 全局 Token 设置 | 管理员配置用于调用 GitLab API 的具有特定只读权限的 Token。 | P0 (核心依赖) |
| **全局导航** | 跨项目全文搜索 | 在顶部搜索框输入关键词，调用 GitLab API 搜索所有白名单仓库内的文件内容及名称，并高亮展示匹配片段。 | P0 (核心体验) |
| **文件树浏览** | 仓库列表展示 | 左侧资源管理器第一层级展示已配置的所有可见仓库。 | P0 (核心体验) |
| **文件树浏览** | 目录懒加载 | 点击仓库或文件夹时，异步请求并展开子目录结构。 | P0 (核心体验) |
| **文件树浏览** | 文件类型图标 | 根据文件后缀（如 `.md`, `.html`, 文件夹等）展示不同的小图标。 | P1 (体验优化) |
| **内容阅读区** | Markdown 渲染 | 右侧主视图将请求到的 Raw Markdown 转换为排版良好的 HTML 页面。 | P0 (核心体验) |
| **内容阅读区** | Mermaid 图表解析 | 识别 Markdown 中的 ````mermaid` 块，并使用 Mermaid.js 渲染为可视化图表。 | P0 (核心体验) |
| **内容阅读区** | HTML 单文件预览 | 右侧主视图通过 iframe 隔离渲染单文件 HTML，实现原型的所见即所得。 | P0 (核心体验) |
| **内容阅读区** | 新标签页全屏预览 | 提供按钮，允许将当前 HTML 原型或 Markdown 文档在新浏览器标签页中全屏打开。 | P1 (体验优化) |
| **深层链接** | URL 状态同步 | 随着左侧树的点击和右侧内容的切换，浏览器 URL 实时更新（包含仓库、分支、文件路径等参数），方便直接复制分享。 | P0 (核心体验) |
