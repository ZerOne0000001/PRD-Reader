# 页面结构 (Page Structure)

## 1. 全局导航与结构

### 1.1 页面清单

| 页面 | 文件 | 说明 |
|:---|:---|:---|
| 首页 | `home.html` | 仓库列表入口，显示当前平台的仓库 |
| 阅读页 | `reader.html` | 左侧文件树 + 右侧阅读区 |
| 后台配置页 | `admin.html` | 系统配置，包括平台选择、Token、白名单 |

### 1.2 路由映射

| URL 路由 | 页面 | 说明 |
|:---|:---|:---|
| `/` | `index.html` / `Home.tsx` | 首页，仓库列表 |
| `/reader` | `Reader.tsx` | 阅读页，文件树 + 阅读区 |
| `/admin` | `Admin.tsx` | 后台配置，需密码验证 |

### 1.3 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                      PRD-Reader                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────┐    ┌─────────────┐    ┌─────────────────┐   │
│  │ Home    │───▶│ Reader      │───▶│ Admin           │   │
│  │ (首页)   │    │ (阅读页)     │    │ (后台配置)       │   │
│  └─────────┘    └─────────────┘    └─────────────────┘   │
│       │              │                    │                │
│       │              │                    │                │
│       ▼              ▼                    ▼                │
│  仓库列表加载     文件树 + 阅读区      平台配置 + Token    │
│       │              │                    │                │
│       │              │                    │                │
│       ▼              ▼                    ▼                │
│  config.json ←── /api/config ←── configStore (Zustand)    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 组件交互说明

| 区域 | 组件 | 交互行为 |
|:---|:---|:---|
| **Header** | 返回按钮 | 点击返回首页 |
| **Tab 页签** | GitLab Tab | 点击切换到 GitLab 配置区，表单内容切换 |
| **Tab 页签** | GitHub Tab | 点击切换到 GitHub 配置区，表单内容切换 |
| **Token 配置** | 实例地址输入框 | 输入 GitHub 实例地址，默认值 `https://github.com` |
| **Token 配置** | Token 输入框 | 输入密码掩码，支持显示/隐藏切换 |
| **Token 配置** | 眼睛图标 | 点击切换 Token 显示/隐藏状态 |
| **Token 配置** | 测试连接按钮 | 点击调用 `/api/config/test`，显示测试结果 |
| **Token 配置** | 保存配置按钮 | 点击调用 `/api/config` POST，保存配置 |
| **白名单** | 添加仓库输入框 | 输入 `owner/repo` 格式，按 Enter 或点击添加 |
| **白名单** | 添加按钮 | 点击添加仓库到白名单 |
| **白名单** | 仓库卡片 | hover 显示删除按钮，点击直接删除 |

### 2.3 状态与反馈

| 交互 | 状态反馈 | 说明 |
|:---|:---|:---|
| 测试连接中 | 按钮显示"测试中..." | 禁用按钮，防止重复点击 |
| 测试连接成功 | 绿色"已连接"状态标识 | 带呼吸动画 |
| 测试连接失败 | 红色"连接失败"文字 | 下方圆角区域显示错误原因 |
| 保存配置成功 | alert 提示"配置保存成功！" | 保存成功后自动刷新白名单 |
| 添加仓库成功 | 列表新增仓库项 | 新增项带入场动画 |
| 添加仓库失败 | alert 显示错误信息 | 仓库已存在/无权限/不存在 |
| 删除仓库 | 直接从列表移除 | 无二次确认 |

### 3.2 组件交互说明

| 区域 | 组件 | 交互行为 |
|:---|:---|:---|
| **左侧文件树** | 抽屉触发区 | 鼠标靠近左侧 40px 边缘自动滑出 |
| **左侧文件树** | 固定按钮 | 点击固定文件树，防止自动收起 |
| **左侧文件树** | 仓库节点 | 显示 ☁️ 图标，点击展开目录 |
| **左侧文件树** | 文件夹节点 | 显示 📁 图标 + Chevron 箭头，点击展开/折叠 |
| **左侧文件树** | 文件节点 | 根据类型显示不同图标，点击打开 |
| **左侧文件树** | 复制地址 | hover 显示复制图标，点击复制路径 |
| **右侧阅读区** | Markdown 内容 | 渲染 Markdown + Mermaid 图表 |
| **右侧阅读区** | TOC 目录 | 右侧悬浮显示，点击跳转章节 |
| **顶部导航** | 菜单按钮 | 鼠标靠近显示，点击展开左侧文件树 |

## 6. 样式与交互规范

### 6.1 样式规范

| 规范 | 说明 |
|:---|:---|
| **设计风格** | 极简清新风（Notion 风格） |
| **圆角** | 大区块 24px，按钮/输入框 12px |
| **阴影** | 卡片使用 `card-shadow` |
| **颜色** | 沿用现有 CSS 变量（`--accent-blue`, `--accent-pink` 等） |
| **隐藏滚动条** | 全局强制隐藏滚动条样式（保留功能） |

### 6.2 交互规范

| 规范 | 说明 |
|:---|:---|
| **Tab 切换** | 即时切换，无加载动画 |
| **表单提交** | 禁用按钮 + loading 状态 |
| **删除操作** | 无二次确认，直接执行 |
| **错误提示** | 使用 alert 或内联错误文字 |
| **成功反馈** | alert 提示或内联成功状态 |
