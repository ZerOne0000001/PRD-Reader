# 用户旅程地图 (User Journey Map)

## 旅程 1: 系统管理员配置 GitHub 平台

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **进入后台** | 点击首页底部入口，输入密码进入后台配置页面 | 验证密码后跳转到配置页面，加载当前配置 | 后台入口按钮、密码输入框 | 提供快捷入口，减少操作步骤 |
| **选择平台** | 在平台选择器中选择"GitHub" | 切换到 GitHub 配置区域，加载已保存的 GitHub 配置（如有） | 平台选择器（单选按钮/下拉框） | 显示平台 Logo 增强识别度 |
| **配置 Token** | 输入 GitHub 实例地址（默认 github.com）和 Personal Access Token | 实时校验输入格式，提供密码显示/隐藏切换 | 实例地址输入框、Token 输入框、显示/隐藏按钮 | 提供 Token 权限说明链接 |
| **测试连接** | 点击"测试连接"按钮 | 调用 GitHub API 验证 Token 有效性，显示测试结果 | 测试连接按钮、状态指示器 | 测试成功显示绿色"已连接"标识 |
| **保存配置** | 点击"保存配置"按钮 | 将配置持久化保存到服务端，显示保存成功提示 | 保存配置按钮、成功提示 | 保存后自动刷新白名单区域 |
| **添加仓库** | 在白名单区域输入仓库路径（如 `owner/repo`），点击添加 | 验证仓库存在性和访问权限，添加到白名单列表 | 仓库输入框、添加按钮、仓库列表 | 支持批量添加，显示仓库详细信息 |
| **验证前台** | 返回前台首页，验证 GitHub 仓库是否可见 | 加载 GitHub 仓库列表，展示文件树 | 前台首页、文件树组件 | 提供快速跳转到前台的链接 |

---

## 旅程 2: 系统管理员切换平台（GitLab → GitHub）

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **进入后台** | 进入后台配置页面 | 加载当前配置，显示当前选中的平台 | 后台配置页面 | 显示当前平台状态标识 |
| **切换平台** | 从 GitLab 切换到 GitHub | 保存当前平台选择，加载 GitHub 的配置和白名单 | 平台选择器 | 切换时显示加载动画，避免闪烁 |
| **验证配置** | 查看 GitHub 配置是否正确 | 显示已保存的 GitHub Token（脱敏显示）和仓库白名单 | 配置展示区域 | Token 脱敏显示，保护安全 |
| **返回前台** | 返回前台首页 | 前台加载 GitHub 仓库列表 | 前台首页 | 提供平台状态指示器 |

---

## 旅程 3: PM 查阅 GitHub 上的 PRD 文档

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **打开首页** | 打开 PRD-Reader 首页 | 静默加载当前配置平台的仓库列表 | 首页 URL | 显示当前平台标识（GitLab/GitHub） |
| **选择仓库** | 点击目标仓库名称 | 展开该仓库的文件树 | 仓库列表、文件树组件 | 支持搜索仓库快速定位 |
| **浏览目录** | 展开目录层级，定位目标文件 | 异步加载目录内容，显示文件图标 | 文件树节点、展开/折叠按钮 | 记住展开状态，下次自动展开 |
| **打开文档** | 点击 `.md` 文件 | 请求文件内容，渲染 Markdown 和 Mermaid 图表 | 文件节点、文档渲染区 | 自动生成 TOC 目录 |
| **阅读文档** | 滚动阅读，点击 TOC 跳转 | 平滑滚动到对应章节 | TOC 目录、文档内容区 | 支持全屏阅读模式 |
| **分享链接** | 复制当前页面 URL 分享给同事 | URL 包含文件路径信息，同事打开即看 | 浏览器地址栏、分享按钮 | 生成短链接便于分享 |

## 旅程 4: 技术人员预览 GitHub 上的 HTML 原型

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **定位文件** | 在文件树中找到 `.html` 原型文件 | 显示 HTML 文件图标 | 文件树组件 | 支持文件类型筛选 |
| **预览原型** | 点击 HTML 文件 | 在新标签页或 iframe 中渲染 HTML 内容 | 文件节点、预览区域 | 提供全屏预览选项 |
| **交互测试** | 在原型中进行交互操作 | 原型交互正常响应 | 预览区域 | 隔离渲染环境，避免样式冲突 |
| **返回文件树** | 关闭预览或切换文件 | 返回文件树，继续浏览 | 关闭按钮、文件树 | 支持多文件同时预览（Tab 形式） |

---

## 旅程 5: 处理平台连接异常

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **发现异常** | 打开首页，发现文件树加载失败 | 显示错误提示："连接失败，请联系管理员检查配置" | 文件树区域、错误提示 | 提供刷新重试按钮 |
| **通知管理员** | 联系系统管理员 | - | 通讯工具 | 系统自动发送告警通知给管理员 |
| **管理员排查** | 管理员进入后台，查看配置状态 | 显示 Token 状态（有效/失效）和错误详情 | 后台配置页面 | 提供详细的错误日志 |
| **修复配置** | 管理员更新 Token 或检查权限 | 保存新配置，测试连接成功 | 配置表单、测试按钮 | 提供一键重新授权功能 |
| **验证修复** | 用户刷新页面 | 文件树正常加载 | 刷新按钮 | 显示"连接已恢复"提示 |
