# 功能清单 (Feature List)

## 1. 功能模块总览

本功能清单基于 `Requirement_Background.md`、`User_Stories.md` 和 `Concept_Design.md` 编制，定义了 GitHub 集成功能的所有功能点。

---

## 2. 功能清单详情

### 2.1 平台配置模块

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
|:---|:---|:---|:---:|
| **平台选择** | Tab 页签切换 | 在后台配置页面提供 GitLab 和 GitHub 两个 Tab，支持点击切换 | P0 |
| **平台选择** | 当前平台指示 | 显示当前选中的平台 Tab 高亮状态 | P0 |
| **平台选择** | 配置自动保存 | 切换 Tab 时自动保存当前平台的配置 | P1 |
| **GitHub 配置** | 实例地址输入 | 提供 GitHub 实例地址输入框，默认值 `https://github.com` | P0 |
| **GitHub 配置** | Token 输入 | 提供密码掩码的 Token 输入框，支持显示/隐藏切换 | P0 |
| **GitHub 配置** | 测试连接 | 提供"测试连接"按钮，验证 Token 有效性 | P0 |
| **GitHub 配置** | 保存配置 | 提供"保存配置"按钮，持久化保存到 config.json | P0 |
| **GitHub 配置** | 连接状态显示 | 测试后显示连接状态（成功/失败），失败显示错误原因 | P0 |

### 2.2 仓库白名单模块

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
|:---|:---|:---|:---:|
| **白名单展示** | 动态过滤 | 根据当前选中的平台 Tab，显示对应平台的仓库列表 | P0 |
| **白名单展示** | 仓库卡片 | 显示仓库名称、路径、状态（正常/异常） | P0 |
| **白名单展示** | 状态标签 | 用颜色区分仓库状态（绿色正常/红色异常） | P1 |
| **添加仓库** | 输入验证 | 支持 `owner/repo` 格式输入 | P0 |
| **添加仓库** | 仓库验证 | 添加时调用 API 验证仓库存在性和可访问性 | P0 |
| **添加仓库** | 重复检测 | 防止重复添加同一仓库 | P1 |
| **删除仓库** | Hover 显示删除 | 鼠标悬停时显示删除按钮 | P0 |
| **删除仓库** | 直接删除 | 点击删除按钮直接从列表移除，无二次确认 | P0 |

### 2.3 文件浏览模块

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
|:---|:---|:---|:---:|
| **文件树** | 仓库列表加载 | 首页加载当前平台的仓库列表 | P0 |
| **文件树** | 目录展开 | 点击仓库或文件夹节点展开/折叠子节点 | P0 |
| **文件树** | 无限嵌套 | 支持多层级目录结构 | P0 |
| **文件树** | 节点图标 | 根据类型显示不同图标（仓库/文件夹/MD/HTML/图片） | P0 |
| **文件树** | 加载状态 | 异步加载时显示 loading 动画 | P1 |
| **文件树** | 错误处理 | 加载失败时显示友好错误提示 | P0 |
| **文件树** | 选中高亮 | 当前阅读的文件节点高亮显示 | P0 |

### 2.4 内容渲染模块

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
|:---|:---|:---|:---:|
| **Markdown 渲染** | 基础渲染 | 渲染 Markdown 为 HTML | P0 |
| **Markdown 渲染** | Mermaid 图表 | 识别并渲染 Mermaid 代码块为 SVG | P0 |
| **Markdown 渲染** | TOC 目录 | 自动生成文档目录并悬浮显示 | P0 |
| **Markdown 渲染** | 代码高亮 | 代码块语法高亮显示 | P1 |
| **Markdown 渲染** | GFM 支持 | 支持 GitHub Flavored Markdown 语法 | P1 |
| **HTML 预览** | iframe 隔离渲染 | 使用 iframe 隔离渲染 HTML 原型 | P0 |
| **HTML 预览** | 全屏预览 | 支持 HTML 原型全屏预览 | P1 |
| **图片查看** | 直接渲染 | 支持 PNG/JPG/SVG/WEBP 直接展示 | P0 |
| **图片查看** | 缩放操作 | 支持图片缩放和拖拽 | P1 |

### 2.5 异常处理模块

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
|:---|:---|:---|:---:|
| **错误提示** | Token 失效 | Token 失效时显示友好提示 | P0 |
| **错误提示** | 权限不足 | 权限不足时显示具体错误 | P1 |
| **错误提示** | 网络异常 | 网络故障时显示重试提示 | P1 |
| **错误提示** | 文件不存在 | 文件不存在时显示提示 | P1 |

## 3. 功能优先级汇总

### P0 - 核心功能（必须实现）

| 模块 | 功能点 |
|:---|:---|
| 平台配置 | Tab 页签切换、Token 输入、测试连接、保存配置 |
| 白名单管理 | 动态过滤、仓库卡片、添加验证、删除功能 |
| 文件浏览 | 仓库加载、目录展开、节点图标、错误处理 |
| 内容渲染 | Markdown 渲染、Mermaid 图表、TOC 目录、HTML 隔离渲染 |

### P1 - 重要功能（应该实现）

| 模块 | 功能点 |
|:---|:---|
| 平台配置 | 配置自动保存、状态标签 |
| 白名单管理 | 重复检测 |
| 文件浏览 | 加载状态 |
| 内容渲染 | 代码高亮、GFM 支持、图片缩放 |
| 异常处理 | 权限不足、网络异常、文件不存在 |

## 4. 功能依赖关系

```text
平台选择 Tab 切换
    │
    ├──▶ GitHub 配置表单显示
    │        │
    │        ├──▶ Token 输入 → 测试连接 → 保存配置
    │        │
    │        └──▶ 仓库白名单动态过滤
    │                 │
    │                 ├──▶ 添加仓库 → API 验证
    │                 │
    │                 └──▶ 删除仓库
    │
    └──▶ 前台文件树加载
             │
             ├──▶ 目录展开
             │
             └──▶ 内容渲染
                      │
                      ├──▶ Markdown + Mermaid
                      │
                      ├──▶ HTML iframe
                      │
                      └──▶ 图片
```
