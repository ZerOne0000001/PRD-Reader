# 功能清单 (Feature List) - F2 文档查询功能

## 1. 功能模块概览

| 一级模块 | 二级功能 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| **S1. 搜索入口** | S1.1 搜索框启用 | 启用 Reader.tsx 顶部导航栏的搜索框，移除 disabled 状态 | P0 |
| | S1.2 搜索触发 | 点击搜索按钮跳转至 /search?q=关键词 | P0 |
| | S1.3 关键词回显 | 搜索结果页顶部搜索框回显当前关键词 | P0 |
| **S2. 搜索结果页** | S2.1 结果统计 | 显示"共找到 X 个结果" | P0 |
| | S2.2 结果列表 | 展示匹配的文件列表（仓库名、路径、文件名） | P0 |
| | S2.3 内容预览 | 对 .md 文件展示内容片段预览（前后各 50 字符） | P0 |
| | S2.4 关键词高亮 | 在预览片段中高亮匹配关键词 | P0 |
| | S2.5 匹配类型标签 | 标注"文件名匹配"或"内容匹配" | P1 |
| **S3. 筛选功能** | S3.1 仓库多选 | 左侧面板支持勾选要搜索的仓库（默认全选） | P0 |
| | S3.2 匹配类型筛选 | 支持按"文件名匹配"或"内容匹配"筛选 | P1 |
| | S3.3 实时过滤 | 筛选条件变更时实时更新结果列表 | P0 |
| **S4. 分页功能** | S4.1 分页控件 | 上一页/下一页/页码按钮 | P0 |
| | S4.2 页码高亮 | 当前页码高亮显示 | P0 |
| | S4.3 状态联动 | 切换页码后保持筛选条件 | P0 |
| **S5. 结果交互** | S5.1 新标签页打开 | 点击结果在新标签页打开 Reader | P0 |
| | S5.2 自动定位 | Reader 页面自动展开文件树并高亮目标文件 | P0 |
| **S6. 状态反馈** | S6.1 加载状态 | 搜索过程中显示加载动画 | P0 |
| | S6.2 空结果状态 | 无结果时展示空状态插图和提示 | P0 |
| | S6.3 错误处理 | 搜索失败时展示错误提示和重试按钮 | P0 |
| **S7. 后端搜索 API** | S7.1 GitLab Search API | 调用 GitLab 私有部署版的 Search API | P0 |
| | S7.2 白名单过滤 | 仅返回白名单仓库内的搜索结果 | P0 |
| | S7.3 结果格式化 | 将 GitLab 返回结果格式化为统一结构 | P0 |

---

## 2. 功能优先级说明

### P0（必须实现）
- S1.1, S1.2, S1.3：搜索入口完整可用
- S2.1, S2.2, S2.3, S2.4：搜索结果核心展示
- S3.1, S3.3：仓库筛选基本功能
- S4.1, S4.2, S4.3：分页功能
- S5.1, S5.2：结果点击跳转
- S6.1, S6.2, S6.3：状态反馈
- S7.1, S7.2, S7.3：后端搜索 API

### P1（优化增强）
- S2.5：匹配类型标签（提升结果可读性）
- S3.2：匹配类型筛选（精细化筛选）

## 5. 验收检查清单

| 功能点 | 验收标准 |
| :--- | :--- |
| 搜索入口 | 搜索框可输入，点击按钮跳转 /search?q=关键词 |
| 结果统计 | 显示"共找到 X 个结果" |
| 结果列表 | 每条结果包含：仓库名、文件路径、文件名 |
| 内容预览 | .md 文件展示前后各 50 字符的预览片段 |
| 关键词高亮 | 预览片段中的关键词使用高亮样式 |
| 仓库筛选 | 左侧面板显示所有仓库，默认全选，可单独勾选 |
| 分页 | 支持上一页/下一页/页码切换 |
| 结果点击 | 在新标签页打开 Reader 并加载对应文件 |
| 加载状态 | 搜索过程显示加载动画 |
| 空结果 | 无结果时显示空状态提示 |
| 错误处理 | 失败时显示错误提示和重试按钮 |
