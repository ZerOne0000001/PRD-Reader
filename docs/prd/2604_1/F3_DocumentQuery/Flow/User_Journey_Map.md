# 用户旅程地图 (User Journey Map) - F2 文档查询功能

## 场景：跨仓库文档检索

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 进入搜索** | 用户在 Reader 页面，鼠标移至顶部展开导航栏 | 导航栏滑出，显示搜索框 | 顶部导航栏 | 可考虑增加快捷键 `Cmd/Ctrl + K` 快速聚焦搜索框 |
| **2. 输入关键词** | 用户在搜索框中输入关键词 | 搜索框实时显示输入内容 | 搜索输入框 | 可考虑输入时显示搜索建议（本次不实现） |
| **3. 触发搜索** | 用户点击搜索按钮 | 按钮显示加载状态，页面跳转至搜索结果页 | 搜索按钮 | 可考虑支持回车键触发搜索 |
| **4. 等待结果** | 用户等待搜索完成 | 搜索结果页显示加载动画 | 搜索结果页面 | 加载动画可展示"正在搜索 X 个仓库..."增加反馈感 |
| **5. 浏览结果** | 用户浏览搜索结果列表 | 展示匹配结果，包含文件路径、内容预览、高亮关键词 | 搜索结果列表 | 可考虑增加筛选器（按仓库筛选） |
| **6. 分页浏览** | 用户点击分页控件 | 切换到对应页面，展示下一批结果 | 分页控件 | 可考虑支持"加载更多"无限滚动模式 |
| **7. 选择结果** | 用户点击某条搜索结果 | 在新标签页打开 Reader 页面，加载对应文档 | 搜索结果项 | 可考虑增加"预览"按钮，在侧边栏快速预览 |
| **8. 阅读文档** | 用户在新标签页阅读文档 | Reader 页面展示文档内容，左侧文件树高亮选中文件 | Reader 页面 | 可考虑滚动到关键词首次出现的位置 |

## 场景：空结果处理

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 输入关键词** | 用户输入一个较为罕见的关键词 | 搜索框显示输入内容 | 搜索输入框 | - |
| **2. 触发搜索** | 用户点击搜索按钮 | 跳转至搜索结果页 | 搜索按钮 | - |
| **3. 查看空状态** | 用户看到空结果提示 | 展示空状态插图和"未找到相关文档"提示 | 搜索结果页面 | 可提供"搜索建议"或"热门文档"推荐 |
| **4. 重新搜索** | 用户修改关键词重新搜索 | 返回步骤 1 | 搜索框 | 可考虑在空状态页面直接提供搜索框 |

## 场景：文件名快速定位

| 阶段 | 用户行为 | 系统响应 | 触点 | 机会点 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 输入文件名** | 用户输入文件名的部分关键词 | 搜索框显示输入内容 | 搜索输入框 | - |
| **2. 触发搜索** | 用户点击搜索按钮 | 跳转至搜索结果页 | 搜索按钮 | - |
| **3. 查看结果** | 用户看到文件名匹配的结果列表 | 结果标注"文件名匹配"标签，排在前列 | 搜索结果列表 | 可考虑文件名匹配结果使用不同样式区分 |
| **4. 选择结果** | 用户点击目标文件 | 在新标签页打开文档 | 搜索结果项 | - |

## 关键触点汇总

| 触点 | 类型 | 交互方式 | 状态反馈 |
| :--- | :--- | :--- | :--- |
| 搜索输入框 | 输入控件 | 键盘输入 | 实时显示输入内容 |
| 搜索按钮 | 触发控件 | 鼠标点击 | 加载状态、禁用状态 |
| 搜索结果列表 | 信息展示 | 鼠标悬停、点击 | 悬停高亮、点击跳转 |
| 分页控件 | 导航控件 | 鼠标点击 | 当前页高亮、加载状态 |
| 空状态页面 | 信息展示 | - | 提示文案、引导按钮 |
