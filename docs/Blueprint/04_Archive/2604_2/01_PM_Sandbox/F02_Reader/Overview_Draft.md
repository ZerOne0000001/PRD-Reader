# 📖 模块概述增量草案：阅读区功能 (F02_Reader)

> 本文件是基于迭代 2604_2（分享文档链接）提取的产品沙箱草案，供开发实现时参考。

## 1. 核心业务价值 (增量)
*   **分享文档链接**：解决目前系统只能分享根域名的痛点。实现深度的 URL 状态绑定与一键分享机制。让开发或测试通过 URL 参数，即可“开箱即用”地直达具体的仓库文件或文件夹，彻底消除寻找成本。

## 2. 关键能力清单 (增量)
*   **深度链接解析 (Deep Linking)**：初始化根据 URL 的 `?repo=&file=` 参数拉取对应仓库树，定位节点并渲染。
*   **下钻模式 (Drill-down)**：基于特定分享链接，屏蔽无关层级，以特定文件夹为根节点进行局部渲染。
*   **气泡选项菜单 (Popover Menu)**：增强左侧文件树操作，将“复制”抽象为弹出菜单，支持 Git 原地址与系统内分享链接。
*   **原位异常拦截 (In-place 404)**：无效或无权限链接触发时，在主阅读组件内部直接接管异常渲染，不触发页面跳转。
