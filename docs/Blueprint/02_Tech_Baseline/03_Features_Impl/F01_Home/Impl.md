# 🏠 首页导流模块实现细节 (F01_Home)

## 1. 代码文件级映射 (Traceability Matrix)

| 职责域 | 核心文件路径 | 核心作用与说明 |
| :--- | :--- | :--- |
| **前端视图 (React)** | `src/pages/Home.tsx` | 独立仓库列表页组件，负责展示网格仓库卡片，引导用户进入阅读模式。 |
| **状态机 (Zustand)** | `src/store/configStore.ts` | 提供系统配置数据（`config.repositories`），作为卡片渲染的数据源。 |

## 2. 核心功能逻辑

### 2.1 状态判断与缺省页展示
*   **流转**：
    1. 用户访问根路径 `/` 且没有 `?repo=` 参数时渲染 `Home.tsx`。
    2. 页面加载时触发 `fetchConfig` 获取系统状态。
    3. 如果系统**未配置**或**白名单为空**：
       * 居中展示缺省提示文案：“暂无可用工作区，请联系管理员配置”。
    4. 如果系统**已配置且存在白名单**：
       * 遍历 `config.repositories`，渲染响应式的卡片网格。
       * 卡片点击后跳转至 `/?repo=${repo.id}`。

### 2.2 管理员入口
*   **交互**：
    *   在页面右上角提供一个固定的齿轮 (`lucide-settings`) 入口。
    *   跳转至 `/admin` 后再进行密码校验与配置。
