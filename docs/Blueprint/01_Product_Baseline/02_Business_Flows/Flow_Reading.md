# 📖 流程：查阅与渲染闭环 (Flow_Reading)

## 1. 流程描述
本流程定义了只读用户如何浏览仓库树并查阅具体的文件内容（Markdown, HTML, 图片等）。

## 2. 步骤拆解
1.  **用户访问首页**：前台用户无需登录，直接打开系统首页。
2.  **静默加载白名单**：系统静默使用全局 Token 读取配置好的仓库白名单。
3.  **浏览文件树**：用户通过左侧抽屉式树状图，像操作本地资源管理器一样浏览仓库和目录（支持无限嵌套展开）。
4.  **内容阅读与渲染**：
    *   **点击 `.md` 文件**：前端请求 Raw 数据，经过 Markdown 和 Mermaid 解析器，在右侧渲染精美排版，并生成独立 TOC (悬浮目录)。
    *   **点击 `.html` 文件**：请求 Raw 数据，在新页面或 `iframe` 中构建隔离视图，实现高保真原型预览。
    *   **点击 图片 文件**：支持 PNG/JPG/SVG/WEBP 等直接渲染展示。

## 3. 业务约束
*   **大文件策略**：针对 `.html` 或图片等非文本文件，后端应提供类似虚拟静态服务器的路由（如 `/api/gitlab/raw/*`），让前端通过 URL 直接加载，而非将巨大内容拉入前端内存。
*   **URL 同步**：阅读内容时，URL 必须随文件路径变化，以支持复制 URL 一键分享给其他用户。
