# 🎨 前端交互与渲染名词 (Domain_UI)

| 中文术语 | 英文术语 (代码中常用) | 释义与语境 |
| :--- | :--- | :--- |
| **主阅读区** | Main Reader Area | 屏幕中央用于渲染 Markdown 或 HTML 的区域。通常限制了最大宽度以提供舒适的阅读视宽。 |
| **抽屉式文件树** | Drawer File Tree | 隐藏在屏幕左侧，鼠标靠近边缘才会滑出的树状导航组件。 |
| **悬浮目录** | TOC (Table of Contents) | 仅在阅读 Markdown 文件时，在屏幕右侧生成的页面大纲，提取自 H1/H2 标签。 |
| **下钻导航** | Drill-down Navigation | 在文件树中，将某个子文件夹提升为临时“根目录”的交互模式，以便专注阅读（该功能目前在设计中提到，前端暂未完全实现）。 |
