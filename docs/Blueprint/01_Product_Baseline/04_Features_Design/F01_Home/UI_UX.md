# 🎨 UI/UX：首页与全局导流 (F01_Home)

## 1. 初始加载与状态反馈
*   **场景一：系统未初始化或无可见仓库**
    *   在页面居中展示缺省提示图及文案：“暂无可用工作区，请联系管理员配置”。
    *   提供引导前往后台配置的入口。
*   **场景二：系统已配置且有可见仓库**
    *   展示系统 Title 及副标题（如“请选择要阅读的工作区”）。
    *   以响应式网格布局 (Grid) 展示所有白名单仓库的卡片。
    *   **仓库卡片交互**：每张卡片包含带首字母的彩色头像（支持 Hover 投影交互与轻微放大）、仓库名称、路径以及默认分支标识。点击卡片后跳转进入对应的阅读区。

## 2. 交互细节
*   右上角固定一个设置按钮，支持 Hover 旋转动画，点击可唤起管理员验证并进入后台。
*   采用 `Soft & Pastel` 主题，背景采用柔和浅色，卡片采用大圆角设计。
