# ⚖️ 业务规则：首页与全局导流 (F01_Home)

## 1. 路由拦截与分发机制
*   **入口判断**：当用户访问根路径 `/` 时：
    *   **存在仓库参数**：若 URL 包含 `?repo=xxx`，则渲染 `Reader` 组件，并将 URL 控制权交给阅读区。
    *   **无仓库参数**：若 URL 不包含 `?repo=xxx`，则渲染 `Home` 独立列表页。

## 2. 后台设置入口权限
*   **入口展示**：首页及阅读区的侧边栏均固定展示“后台配置”入口。
*   **鉴权逻辑**：点击进入前需输入系统统一密码（硬编码 `Aa@000000`）。错误时拦截跳转并报错，正确时允许跳转至 `/admin`。
