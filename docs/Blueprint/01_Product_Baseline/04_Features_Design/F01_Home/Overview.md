# 🏠 功能概述：首页与全局导流 (F01_Home)

## 1. 业务目标
处理应用流量入口的初始加载判断与仓库选择。作为用户接触系统的第一站，必须做到极简、清晰，提供直观的仓库列表，并在后台未配置时给出明确指引。

## 2. 核心价值
*   **状态感知**：静默判断系统是否已完成初始化配置（Token、白名单）。
*   **独立仓库列表 (Repository Hub)**：根据系统状态，将已配置的用户直接展示所有可用仓库的卡片网格；将未配置的用户导向后台配置入口。
*   **降低认知负荷**：通过独立的首页展示仓库卡片，让用户在进入具体文件树前先选择工作区，避免多个仓库文件树混合展示带来的视觉混乱。
*   **Deep Linking (深度链接) 支持**：若用户直接携带 `?repo=` 参数访问，系统将静默渲染对应的文件树，而不是停留在首页。
