# 🎨 UI/UX：阅读区功能 (F02_Reader)

## 1. 布局结构 (Layout)
*   **左侧抽屉式侧边栏 (Sidebar Drawer)**：
    *   **下拉仓库切换器**：顶部包含当前仓库名称与下拉箭头，点击弹出悬浮菜单（Popover）进行快速仓库切换。
    *   **单仓库文件树**：下方展示该仓库下的文件树，支持折叠/展开，超长文字悬浮提示 (Smart Tooltip)。
    *   **分享交互区**：鼠标悬停在文件树节点时出现“分享”图标，点击弹出小型气泡菜单，提供“复制系统链接”和“复制 Git 地址”并附带视觉反馈。
    *   **固定与隐藏**：默认可被鼠标滑出，可通过顶部的大头针 (`lucide-pin`) 按钮固定在页面左侧。
*   **右侧主内宽区 (Main Reading Area)**：
    *   **顶部悬浮导航 (Header)**：展示当前文件的完整面包屑路径。同样支持滑出与固定。
    *   **中央阅读容器**：限制最大阅读宽度（`max-w-3xl`），保证行长适中。背景色与卡片阴影采用 `Soft & Pastel` 规范。
    *   **右侧目录 (TOC)**：仅在阅读 `.md` 文件且有内宽时，在屏幕最右侧固定展示提取的标题目录。

## 2. 文件渲染样式
*   **Markdown (`.md`)**：
    *   H1 采用带黄色高亮下划线的 `Quicksand` 字体。
    *   H2 带有粉色花朵前缀 `✿`。
    *   代码块、表格、列表等元素被美化并具有统一的柔和边框和圆角。
    *   内置支持 `mermaid` 语法的类 UMD 实时渲染。
*   **原型文件 (`.html`)**：
    *   占满主内宽区（无额外 padding），使用 `iframe` 渲染，设置合理的安全策略 (`sandbox`)。
    *   在左侧文件树中，该类型文件提供“在新标签页打开”(`lucide-monitor`) 快捷按钮。
*   **图片**：在浅灰背景中居中显示，包含微弱阴影和圆角，保留其原始比例 (`object-contain`)。

## 3. 异常与边界状态 (In-place 404)
*   **缺省占位符**：当仓库无权限或目标文件不存在时，不触发页面跳转，直接在主内宽区渲染 403 / 404 状态卡片。
*   **加载反馈**：获取文件树或长文档时，主内宽区需显示 Spin 动画，避免用户疑惑。
