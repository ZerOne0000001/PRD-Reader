# 📖 功能概述：阅读区功能 (F02_Reader)

## 1. 业务目标
为非技术人员（产品经理、运营）提供一个干净、直观的沉浸式文件阅读界面，支持跨仓库一键切换，无需理解复杂的 Git 操作即可阅读 `.md` 和 `.html` 原型文件。

## 2. 核心价值
*   **透明的平台适配**：对用户屏蔽底层数据来源于 GitLab 还是 GitHub，提供统一的文件树和文档阅读体验。
*   **沉浸式单仓库阅读**：阅读区仅展示当前选中的单个仓库，避免信息干扰，通过下拉菜单实现跨仓库极速切换。
*   **所见即所得**：Markdown 原生支持，HTML 原型文件在 `iframe` 中安全渲染，支持在新标签页中无边框全屏打开。
*   **智能 TOC 与下钻分享**：自动生成右侧目录，并支持一键复制特定文件夹/文件的深度链接（下钻模式），解决跨部门沟通效率低下的痛点。
